# IDE ( Integrated Development Environment)
- Powerful tool simplifies and speeds up the process of writing and managing code for software development.

# Ket Features
- Code Editor
- Compiler and interpreter
- Debugger
- Build automation : The IDE can automatate tasks as compiling code, packaging software, running unit tests, and deploying applications.
- Version Control Integration
- etc....

# Why use an IDE ?
- Efficiency 
- Error detection
- Convenience
- Productivity
- etc...
