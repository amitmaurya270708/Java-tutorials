# Java

- High level, OOPs language
- Very popular
- Can build robust, secure & scalable applications

# Key features

- Platform independence: Write Once, Run Everywhere due to JVM (Java Virtual Machine).
- Object Oriented
- Robust & secure: Exception handling, garbage collection and security features.
- Multi-threaded
- Rich api - UI, networking, collections, etc.

# Application

- Web development (spring)
- Mobile development (android)
- Enterprise application (banking)
- Big data (hadoops)
- etc...
